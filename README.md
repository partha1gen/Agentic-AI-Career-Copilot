AI Career Copilot

AI Career Copilot is an Agentic AI application built to help candidates throughout their job search journey. It enables users to upload resumes and job descriptions, compare their skills against role requirements, identify gaps, generate personalized learning plans, and ask career-related questions grounded in their uploaded documents.

Key Features
Resume ↔ Job Description analysis with candidate match scoring
Skill gap identification
Personalized learning plan generation
RAG-based document question answering
Multi-turn conversation with persistent session memory
Tool-based orchestration using OpenAI Function Calling
Semantic document retrieval with ChromaDB
